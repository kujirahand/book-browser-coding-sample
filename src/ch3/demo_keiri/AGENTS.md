# 経理サイトから売上データを取得してCSVに保存するプログラムの作成について

## アプリの概要

PythonとPlaywright MCPを使って、経理用のサイトから、各支店の売り上げデータを取得して、CSVファイルに保存します。

## ログインページについて

最初に以下のURLにアクセスして、ログインしてください。

```url
https://api.aoikujira.com/demo_keiri/login.php
```

次のログイン情報を使ってください。なお、ログインページは、JavaScriptで動的に生成されているので、その点を考慮してください。

- ユーザーID: admin
- パスワード: pOuIohVNS4aDOY8i

## 支店一覧ページについて

ログインすると、支店の一覧ページが表示されます。「支店ID」と「支店名」のテーブルがあります。支店IDをクリックすると、その支店の売り上げデータが表示されます。

## 売り上げデータページについて

売り上げデータページには、以下の情報が表示されます。

- 取引ID
- 日時
- 商品ID
- 商品名
- 単価
- 数量
- 小計
- 支払い方法

ページャーもあります。ページャーで全てのページを巡回して、全ての売り上げデータを取得してください。

## CSVファイルへの保存

支店ごとに、売り上げデータをCSVファイルに保存してください。ファイル名は「branch_{支店ID}.csv」としてください。

## プログラムの作成について

プログラムを作成して、上記の機能を実装してください。Playwright MCPを使って、ブラウザを自動で操作して、ログインからデータの取得、CSVファイルへの保存までを行うプログラムを作成してください。

なお、プログラムの役割を明確にするために、次のように役割ごとにモジュールに分割してください。

```text
ch3/demo_keiri/
├── config.py  ... 設定ファイル
├── main.py  ... メイン処理
├── scraper.py ... Playwrightを使ったブラウザ操作の処理
├── parser.py  ... 売上情報のパース処理
└── storage.py ... CSVファイルへの保存処理
```

## 注意点

- エラーが発生した場合は、エラーメッセージをログに記録してください。
- サーバー負荷を軽減するために、ページ遷移をする際は、3秒-5秒の待機時間を入れてください。
