# Image Downloader - シンプルなブラウザ拡張

ページ内の全画像をワンクリックでダウンロードするブラウザ拡張です。

## ファイル構成

- **manifest.json** - 拡張の設定ファイル
- **popup.html** - ポップアップUI
- **popup.js** - ダウンロード処理のメインロジック
- **content.js** - ページ内の画像取得スクリプト

## インストール方法

### Chrome/Edge（開発者モード）

1. `chrome://extensions/` を開く
2. 右上の「開発者モード」をON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択

## 使い方

1. 画像を含むページを開く
2. 拡張アイコンをクリック
3. 「全画像をダウンロード」ボタンをクリック
4. 画像が自動的にダウンロードされます
