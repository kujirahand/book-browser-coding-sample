# 作図専門エージェント

## 概要

このプロジェクトは、作図専門のエージェントを開発することを目的としています。エージェントは、ユーザーの指示に基づいて様々な種類の図を生成することができます。これには、フローチャート、マインドマップ、ネットワーク図などが含まれます。

## 利用技術

- Python + pywebview + Flask
- Mermaid.js
- LiteLLM (OpenAI API)

## 仕様

- 画面上部にプロンプト入力欄と送信ボタン
- 画面レイアウトは3:2で、左側に図形表示画面: 右側にMermaid.jsのコード
- Mermaidのコードは、よくエラーになるので、エラーが出たら、AIで修正するボタンを配置する
