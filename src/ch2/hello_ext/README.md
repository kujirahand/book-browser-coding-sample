# Hello Extension - 格言を表示するブラウザ拡張

クリックするとポップアップにランダムな格言を表示する、シンプルなブラウザ拡張です。

## ファイル構成

- [manifest.json](manifest.json) - 拡張の設定ファイル
- [popup.html](popup.html) - ポップアップUI
- [popup.js](popup.js) - 格言をランダム表示するスクリプト

## インストール方法

### Chrome/Edge（開発者モード）

1. `chrome://extensions/` を開く
2. 右上の「開発者モード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリックする
4. このフォルダを選択する

## 使い方

1. ブラウザのツールバーにある拡張アイコンをクリックする
2. この拡張を選ぶ
3. ポップアップが開き、格言が1つ表示される
4. 開くたびにランダムで別の格言を確認できる

## 実装のポイント

- `popup.js` に格言の配列を定義しています
- ポップアップの読み込み時に、配列から1件をランダムに選んで表示します
- 追加の権限を必要としない最小構成の拡張です
