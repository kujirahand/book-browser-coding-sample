# ainote

ノート保存、検索、要約、作図支援をまとめたAIノートアプリです。Flaskサーバーと補助ディレクトリで構成されています。

- [app.py](app.py) - アプリ起動処理
- [requirements.txt](requirements.txt) - 依存ライブラリ
- [services](services/) - 機能別サービス群
- [static](static/) - フロントエンド
- [data](data/) - インデックスや中間データ
- [text](text/) - ノート本文

実行例:

```bash
pip install -r requirements.txt
python app.py
```
