# book-browser-coding-sample

書籍のサンプルプログラム

## GoやRustが分からない人のためのフォローについて

本書では、ブラウザ・コンポーネントを利用してデスクトップアプリを作る`Wails`や`Tauri`についても解説しています。ただし、これらのフレームワークでは、Go言語やRustなどのプログラミング言語を使います。ただし、これらのフレームワークを使う場合でも、実際には、フロントエンド側は、HTML/CSS/JavaScriptで使います。バックエンドのほんの一部のみをGo言語やRustで記述します。

ただし、Go言語やRustが分からないという人には、ストレスになるかもしれません。書籍は400ページを越えており、本文中では、GoやRustの詳しい解説はできなかったため、このサンプルに解説を含めています。

  - [Tauriでファイルを読み出すプログラムを作ろう](src/ch4/tauri-file-reader/README.md) --- Rustのプログラムの解説
- 5章
