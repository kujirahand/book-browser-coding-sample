# 第1章のプログラム概要

この章では、Pythonを使った最も基本的なサンプルとして `hello.py` を扱います。

## 含まれるプログラム

- `hello.py`: Pythonで標準出力に `Hello, World!` を表示する最小構成のサンプルです。Pythonスクリプトの実行方法と、プログラムが上から順に処理される基本を確認できます。

## 学べること

- Pythonスクリプトの基本的な書き方
- `print()` による文字列の表示
- 開発環境でプログラムを実行する最初の流れ

## 実行例

```bash
cd src/ch1
python hello.py
```

実行すると、ターミナルに `Hello, World!` と表示されます。

## 実行結果の例

```text
Hello, World!
```

## 補足

環境によっては `python3 hello.py` のように実行します。
